# Per-Category Analysis

This directory contains:

- **`_all-45.md`** — single-file summary of all 45 categories: purpose, topology default, anchor automations, required stack, persona fit, deployment notes.
- **`cat-01-lead-generation.md`** — deep dive on Lead Generation & Prospecting (the largest category, 20 automations, T5-heavy).
- **`cat-20-compliance.md`** — deep dive on Compliance & Legal (anchor to PA CROP reference implementation).
- **`cat-38-field-service.md`** — deep dive on Field Service & Dispatch (representative of industry-specific categories).
- **`cat-45-ai-orchestration.md`** — deep dive on AI Agent & Workflow Orchestration (meta-automation; governs other categories).

Deep dives are written for 4 of the 45 categories as working examples. Patterns and schemas established in those four are the templates that deployment operators use to write their own deep dives for the remaining 41 categories on a per-tenant basis. The `_all-45.md` file is the authoritative per-category reference for day-to-day work.
