Common questions

Whose accounts does everything run on?

The client's. You hand off ownership on day 14. No lock-in to us. No shared Stripe accounts, no shared Vercel, no shared domain. Documented in MANUAL-ACTIONS.md shipped with every build.

Can I white-label this completely?

Yes. Your logo, your domain, your pricing. Clients never see "Your Deputy" or parent-company branding. Enable white-label mode in Settings.

What if a module fails during provisioning?

Each integration has a documented manual fallback in the repo. Soft failures don't block the launch — the build completes and the affected module is flagged for manual setup.

How is this different from GoHighLevel or Shopify?

GHL ties clients to their platform permanently. Shopify is one vertical (ecom). We generate code + infrastructure the client fully owns, across any vertical, with all 19 modules integrated and no platform lock-in.

Do I need technical skills to run this?

No. If you can describe a business in a paragraph, you can run a build. For technical edge cases, we provide support on the $4,997+ tiers.

What's the catch I should know about?

Some integration modules require vendor keys that vary by niche (SparkReceipt for bookkeeping, Heartbeat for community). For the first 3 launches, we walk you through which modules to enable for your specific clients. After that, it's self-serve.

Can I resell the Managed tier as my own retainer?

Yes. White-label the Managed ops at any margin. Most agencies charge clients $1,500–$2,500/mo and pay us $497.

What AI models power this?

Free-tier frontier open-weight models — Llama 4, GLM-4.6, DeepSeek R1, GPT-OSS-120B, Qwen3-Coder 480B — routed across 20 providers (Groq, Cerebras, SambaNova, GitHub Models, Cloudflare, etc.). $0 inference cost per build.
