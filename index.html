FAQ
Common questions

Is the generated code production-ready?

Yes. FastAPI with JWT auth, rate limiting, structured error handling, request ID tracking, health checks. Next.js 14 with TypeScript. CI/CD via GitHub Actions, Docker, Vercel with security headers.

What AI models are used?

Models across seven providers — Anthropic, OpenAI, Google, Groq, DeepSeek, Mistral, OpenRouter. Smart routing optimizes for quality and cost per phase.

Do I own the generated code?

100%. Everything is pushed to your private GitHub repository. Full ownership, no attribution required.

What infrastructure gets provisioned?

GitHub repo, Vercel hosting, Neon PostgreSQL, Stripe billing, Acumbamail email, Pulsetic monitoring, n8n automation — all automatically.

How is this different from a template?

Templates are generic. Dynasty generates a complete business system — strategy, financial models, go-to-market playbooks, custom code — all tailored to your specific idea and market.

What happens after the build?

Clone the repo and use Claude Code or Pi Agent. The generated AGENTS.md gives AI full context. Run docker-compose up, complete MANUAL-ACTIONS.md, start acquisition per GTM-PLAYBOOK.md.
