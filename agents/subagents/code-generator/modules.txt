Standing facts:
- Node 20.x is the runtime. ES modules throughout.
- Next.js projects use app router unless the plan specifies pages.
- Python services use FastAPI; avoid Flask for new work.
- Every Vercel-hosted project ships with `/api/health` returning
  `{ok:true, version, commit}`.
- Every project's README ends with a "Deploying" section that names the
  Vercel team and GitHub owner.
- Every n8n workflow exported from this project lives under
  `automations/workflows/` as JSON with a sibling README.
