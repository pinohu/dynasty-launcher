## Deployment Steps

1. Clone this directory to your dynasty-launcher repo
2. Configure environment variables (see each API file header)
3. Import n8n workflows via n8n UI (Settings → Import Workflow)
4. Deploy Vercel APIs via `vercel deploy`
5. Configure Stripe webhooks to point to relevant API endpoints
6. Set up capability integrations (email, SMS, phone, calendar)
7. Run activation flow for each module per tenant
