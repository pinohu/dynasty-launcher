[ ] Set environment variables:
    - VERCEL_API_URL (API base URL)
    - API_KEY (n8n to Vercel authentication)
    - DATABASE_URL (Neon PostgreSQL)
    - ADMIN_EMAIL (for notifications)

[ ] Configure Neon PostgreSQL:
    - Create required tables from schema
    - Set up connection pooling
    - Enable replication if needed

[ ] n8n Setup:
    - Import JSON workflows
    - Configure credentials for external services
    - Test webhooks with sample payloads
    - Enable error notifications

[ ] Vercel Deployment:
    - Deploy Express API to Vercel Functions
